Css bir web sayfasında nasıl kullanılır?

  1. Yerel Css
  2. Dahili/Genel Css
  3. Harici/Bağlantılı Css

Yerel Css

Yerel stil şablonları bir web sayfasında bir html etiketi için özel olarak kullanılırlar.
Yerel Css kullanım örneği bu paragrafın parametresi olarak yer alan style etiketi içinde vardır.

Genel/Dahili Css

Genel css yazıldığı sayfanın tamamında etkilidir.
Yani bu stiller ilgili sayfadaki ilgili etiketlerin hepsinin özelliklerini belirlemek istediğimizde kullanılır.
Yenel Css stil etiketleri head bloğunda style etiketi içine yazılır. Yazılan özellikler sadece ilgili sayfada etkili olacaktır.

Harici/Bağlantılı Css

Harici Css birden çok sayfada kullanılabilir. Birden çok sayfanın aynı stillerde olması istendiğinde kullanılır.
Harici css için .css uzantılı ayrı bir belge oluşturulur. Bu belge head blogunda link etiketi ile sayfaya dahil edilir. örnek bu sayfanın head bloğunda vardır.
Örneğin bu sayfadaki h1 etiketleri harici css sayfasında tanımlanmıştır.

Seçiciler

Seçiciler bir html sayfasında bulunan herhangi bir html elemanını veya birden fazla elamanı seçerek özellik biçimlendirmeyi sağlar.

Sınıf Seçicileri

Sınıf seçicisi birden çok html elemanını biçimlendirmek istediğimizde kullanılır.
.sınıfadi şeklinde css içinde tanımlanır.

Etkilenecek olan html elamının parametresi olarak class="sınıfadi" şeklinde çağırılır. Örnek bu paragraf etiketinde yer almaktadır.

İd Seçicisi

Id seçici bir html elemanının özel biçimlendirmek istediğimizde kullanılır. Css belgesi içinde #idadi şeklinde tanımlanır.
Etkilenecek olan html elemanının parametresiolarak id="idadi" şeklinde çağırılır. Örnek bu paragraf etiketinde yer almaktadır.
